# Snowball Engine

Thin shell over core `snowball-plane`.

Examples:

- `infring snowball start --cycle-id=s17 --drops=core-hardening,app-refine`
- `infring snowball melt-refine --cycle-id=s17 --regression-pass=1`
- `infring snowball compact --cycle-id=s17`
- `infring snowball backlog-pack --cycle-id=s17`
- `infring snowball status --cycle-id=s17`
