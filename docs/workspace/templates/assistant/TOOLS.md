# TOOLS.md - Local Notes

Skills define _how_ tools work. This file is for local specifics that are unique to one setup.

## Bootstrap

- Generated into `local/workspace/assistant/TOOLS.md` by `npm run local:init` if missing.
- Store machine-specific notes here instead of shared skills or tracked docs.
- Treat it as local operator infrastructure, not product documentation.

## What Goes Here

- Camera names and locations
- SSH hosts and aliases
- Preferred voices for TTS
- Speaker or room names
- Device nicknames
- Any environment-specific operational notes

Keep this file local to the instance. Skills are shared; setup notes are not.
