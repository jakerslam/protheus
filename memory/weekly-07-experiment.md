# weekly-07-experiment — Week 7 Experiment Design

**Hypothesis:** Memory-first agents retain 2x context in multi-turn conversations

**Setup:** 
- Condition A: Agent references memory nodes explicitly in replies
- Condition B: Agent responds without memory references
- Track: Engagement rate, conversation depth, follow-up questions

**Metric:** Reply rate + conversation depth (turns)
**Pass Threshold:** 1.5x improvement with memory references
**Duration:** 2 weeks
**Status:** Design complete, awaiting implementation
