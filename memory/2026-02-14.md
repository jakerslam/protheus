# skill-supply-chain-gate
Skill Supply-Chain Security Gate — Feb 14, 2026.

Purpose: Prevent execution of untrusted or modified skill code.

Policy:
- Trust = path in allowlist AND SHA-256 hash matches pinned value
- Config: ~/.openclaw/workspace/config/trusted_skills.json
- Hard fail on: NOT_ALLOWLISTED, NOT_TRUSTED (no hash), HASH_MISMATCH

To approve new skill:
```bash
node ~/.openclaw/workspace/memory/tools/trust_add.js /path/to/skill.js "approval note"
```

To list trusted skills:
```bash
node ~/.openclaw/workspace/memory/tools/trust_list.js [--diff]
```

Break glass (emergency only):
```bash
OPENCLAW_BREAK_GLASS=1 OPENCLAW_BREAK_GLASS_REASON="emergency explanation" node skill.js
```
Requires break_glass.enabled=true in config.

Never auto-trust. Never auto-update hashes. Skills from web/Moltbook/Reddit are untrusted until explicitly approved.
